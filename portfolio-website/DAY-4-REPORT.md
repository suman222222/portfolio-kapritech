# Overview

Day 4 focused on adding more useful JavaScript to the portfolio.

The goal was to move from one basic interaction into practical website features: contact form validation and project filtering.

---

# Major Accomplishments

## Contact Form Added

Built a contact form on the Contact page with fields for:

* Name
* Email
* Message

The form checks whether the visitor entered a valid name, email, and message before preparing an email draft.

## Project Filters Added

Added filter buttons on the Projects page so visitors can sort work by:

* All
* Live
* Learning
* Ideas

This makes the project section easier to scan as KapriTech grows.

## JavaScript Practice Continued

Expanded `script.js` beyond mobile navigation.

New JavaScript concepts practiced:

* Form submit events
* Preventing default browser behavior
* Reading form input values
* Regular expression email validation
* Showing error and success messages
* Filtering elements with data attributes

## Homepage Welcome Improved

Updated the homepage welcome section to feel warmer, clearer, and more personal.

Added:

* Stronger welcome headline
* Clearer introduction copy
* Skill and progress highlight badges
* More polished background and color system

## Message Buttons and Stats Added

Added a homepage JavaScript practice section with:

* Message buttons
* Browser alert button
* KapriTech stats
* DOM text updates
* DOM number updates
