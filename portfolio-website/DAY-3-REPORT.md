# Overview

Day 3 focused on improving the portfolio experience after the Day 2 foundation was complete.

The goal was to make the website more usable on mobile, add the first JavaScript interaction, and continue documenting the public building journey.

---

# Major Accomplishments

## Responsive Design Improvements

Improved the mobile navigation layout so the site is easier to use on small screens.

Added:

* Mobile menu button
* Collapsible navigation links
* Better mobile spacing
* Stronger focus states for keyboard users

## JavaScript Interaction

Created `script.js` and added the first interactive website behavior.

The script:

* Opens and closes the mobile navigation
* Updates the `aria-expanded` state
* Closes the menu after a navigation link is selected

## Project Showcase Improvements

Made project cards clickable so the Projects page feels more like a real portfolio showcase.

Current project cards now link to:

* Portfolio Website
* KapriTech Founder Diary
* Learning Notes

Project cards now behave like links, which makes the Projects page more practical for visitors.
