# CSS Concepts Learned

## 1. External CSS Linking

```html id="4s5mgo"
<link rel="stylesheet" href="style.css">
```

Purpose:
Connects CSS file to HTML.

---

## 2. Selectors

```css id="2qmbi8"
body {
    background: black;
}
```

Purpose:
Targets HTML elements for styling.

---

## 3. Colors

```css id="d8w1jq"
color: white;
background-color: black;
```

Purpose:
Controls text and background appearance.

---

## 4. Margin

```css id="ws9hpb"
margin: 20px;
```

Purpose:
Creates outside spacing.

---

## 5. Padding

```css id="4k7v7y"
padding: 20px;
```

Purpose:
Creates inside spacing.

---

## 6. Flexbox

```css id="jlwm4w"
display: flex;
justify-content: center;
align-items: center;
```

Purpose:
Creates modern layouts and alignment.

---

## 7. Hover Effects

```css id="0j0f7x"
button:hover {
    background: blue;
}
```

Purpose:
Adds interaction when mouse hovers.

---

## 8. Typography

```css id="cwjlwm"

```

Purpose:
Controls text appearance.

---

## 9. Border Radius


Purpose:
Creates rounded corners.

---

## 10. Responsive Preparation

```css id="jlwm5k"

```

Purpose:
Helps layouts adapt to screen sizes.

## 4. Git Push
