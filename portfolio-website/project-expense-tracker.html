Expense Tracker Pro

A practical full‑stack web application for tracking personal expenses, built with HTML, CSS, JavaScript, and Python Flask.

About the Project

Expense Tracker Pro allows users to add, edit, and delete expenses, categorise them, and view total spending. The backend is built with Flask and uses a lightweight SQLite database, while the frontend is a responsive single‑page application.
