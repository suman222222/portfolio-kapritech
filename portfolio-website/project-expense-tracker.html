Day 10 project detail

Expense Tracker Pro

A KapriTech web app for tracking expenses with search, summary cards, category filters, NPR currency, and edit/delete actions while practicing Flask backend thinking.

HTML CSS JavaScript Python Flask Day 10

Purpose

Help users record expenses, review totals, search transactions, and keep a clearer view of where money is going.

Built so far

  • Dark theme interface with a theme toggle
  • Search bar for finding expenses quickly
  • Summary cards for total expenses, transactions, categories, and food items
  • Add-expense form with name, amount, date, category, and currency fields
  • Expense table with edit and delete actions
  • NPR currency display for local context
  • Python Flask app structure and routes

Lessons learned

A useful app needs more than a form. Search, summaries, currency, actions, routes, validation, and data structure all need to work together.

Next improvements

Add stronger persistence, category charts, monthly summaries, validation messages, and a cleaner responsive table for mobile screens.
